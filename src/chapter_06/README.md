## Project Overview

Analyze the Tips dataset to uncover patterns in restaurant tipping behavior. This capstone brings together techniques from all previous chapters into a single cohesive analysis workflow, covering data exploration, manipulation, and visualization.

The project uses the `tips` dataset from Seaborn, which contains 244 records of restaurant transactions including total bill, tip amount, customer sex, smoking status, day of the week, meal time, and party size.

## Skills Practiced

- Loading and inspecting datasets with Pandas
- Computing summary statistics and checking data quality
- Filtering, grouping, and aggregating DataFrames
- Creating derived columns from existing data
- Building pivot tables for cross-tabulated summaries
- Plotting histograms and scatter plots with Matplotlib
- Creating box plots, heatmaps, and FacetGrids with Seaborn
- Interpreting correlation matrices
- Drawing actionable insights from visualizations
