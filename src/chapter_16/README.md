## Project Overview

Classify wine varieties using 13 chemical properties from sklearn's built-in
Wine dataset. This is a multiclass classification problem with 3 wine classes
and features such as alcohol content, malic acid, ash, magnesium, flavanoids,
color intensity, proline, and more.

The project walks through the full ML workflow: exploratory data analysis,
preprocessing, feature engineering with PCA, training and comparing multiple
models, hyperparameter tuning, and final evaluation.

## Skills Practiced

- Data exploration and visualization
- Feature scaling and PCA
- Model training (Logistic Regression, Random Forest, SVM)
- Cross-validation and model comparison
- Hyperparameter tuning with GridSearchCV
- Final evaluation with confusion matrix and classification report
